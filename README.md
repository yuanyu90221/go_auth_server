# go_auth_server

## introduction 

    This is an test project for build an auth server with gin framework

## how to use

### 1 install package
```golang===
go install
```
### 2 setup .env
```yaml===
PORT=${SERVER_PORT}
```

### 3 run
```golang===
go run main.go
```

### 4 build
```golang===
go build
```
